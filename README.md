# 项目介绍
这个项目是一个旨在为Bevy引擎提供完整Mod支持的框架，允许开发者通过简单的宏标记，像编写普通Bevy插件一样开发模组，而主游戏只需一行代码即可加载并运行这些模组。
该系统基于WASM沙箱技术，确保Mod的安全性与跨平台兼容性，同时通过过程宏+描述式注册+Host 代理机制，实现对ECS资源、实体查询、系统调度等核心功能的无缝支持。
